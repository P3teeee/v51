Canvas

html5 element

        <canvas id="canvas" width="450" height="450">Fallback, sad browser</canvas>

För att arbeta med det så behöver vi välja elementet och dess context. 2d

        var canvas = document.getElementById('canvas');
        var ctx = canvas.getContext('2d');

För att rita på canvas så använder vi funktioner, ex rita ut en kub.

        ctx.fillStyle = color;
        ctx.fillRect(x ,y ,sizeX ,sizeY);
