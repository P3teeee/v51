Sass

    Sass använder Ruby
    Installera ruby http://rubyinstaller.org/
    gem install sass
    sass -v

    sass input.scss output.css

    SCSS syntax

    variabler $varName

    selector {
        property: $varName;
    }

    Partials & import

    @import 'filnamn';

    Mixins - små bitar för att göra livet enklare, tänk funktioner (finns också)

    @mixin border-radius($radius) {
      -webkit-border-radius: $radius;
         -moz-border-radius: $radius;
          -ms-border-radius: $radius;
              border-radius: $radius;
    }

    .box { @include border-radius(10px); }

    Operators, standard math 
    width: 300px / 960px * 100%;

    $people: jens, jens, jens;

    @each $person in $people {
        .person-#{$people} {
            background: url(person-#{$people}.jpg);
        }
    }
